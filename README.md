## Descripción

Este repositorio contiene las actividades, ejercicios, prácticas y programas desarrollados durante el curso de Métodos Numéricos.

Los trabajos se encuentran organizados por unidades para facilitar su consulta y evaluación.

## Estructura del repositorio

### Unidad 1
Introducción a los métodos numéricos y actividades desarrolladas en Python.

### Unidad 2
Métodos de solución de ecuaciones no lineales:
-Método de Newton-Raphson.
-Método de Bisección

### Unidad 3
Solución de sistemas de ecuaciones utilizando Jacobi y Gauss-Seidel.

### Unidad 4
Métodos de integración numérica:
- Regla del Trapecio
- Regla de Simpson

### Unidad 5
Interpolación mediante el método de Lagrange.

### Unidad 6
Ecuaciones diferenciales ordinarias:
- Euler
- Euler Mejorado (Heun)
- Runge-Kutta de cuarto orden
- Sistemas de EDOs

## Herramientas utilizadas

- Java
- Python
- Microsoft Excel
- Visual Studio Code
