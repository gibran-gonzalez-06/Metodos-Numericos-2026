# Unidad 2 - Métodos de Solución de Ecuaciones

## Descripción

En esta unidad se estudiaron métodos numéricos para encontrar raíces de ecuaciones no lineales. Se realizaron ejercicios prácticos utilizando herramientas computacionales y se desarrolló un programa para implementar el método de Newton-Raphson.

## Objetivo

Aplicar métodos numéricos para obtener aproximaciones de las raíces de ecuaciones mediante procesos iterativos.

## Actividades realizadas

- Resolución de ejercicios en Microsoft Excel.
- Aplicación del método de Newton-Raphson.
- Comparación de resultados obtenidos mediante iteraciones.
- Desarrollo de un programa en Java para automatizar los cálculos.

## Programa desarrollado

### NewtonRaphson.java

Programa que permite aproximar la raíz de una ecuación mediante el método de Newton-Raphson utilizando iteraciones sucesivas hasta alcanzar la tolerancia establecida.
