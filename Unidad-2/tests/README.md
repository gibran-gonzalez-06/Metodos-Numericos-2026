# Evidencias de Ejecución

## Descripción

Esta carpeta contiene capturas de pantalla que muestran el correcto funcionamiento del programa desarrollado.

## Evidencias

Las pruebas realizadas muestran:

- Valor inicial.
- Evaluación de la función.
- Evaluación de la derivada.
- Iteraciones del método.
- Cálculo del error.
- Obtención de la raíz aproximada.

## Resultado

El programa logra aproximar correctamente la raíz de la función utilizando el método de Newton-Raphson.
