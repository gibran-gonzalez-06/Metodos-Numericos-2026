# Entorno de Ejecución - Unidad 3

## Requisitos

- Java JDK 17 o superior.
- Visual Studio Code o NetBeans.
- Sistema operativo Windows.

## Librerías utilizadas

No se utilizaron librerías externas.

Los programas fueron desarrollados únicamente con Java estándar.

## Ejecución

1. Abrir el archivo correspondiente:
   - Jacobi.java
   - GaussSeidel.java

2. Compilar el programa.

3. Ejecutar el programa.

4. Ingresar:
   - Número de ecuaciones.
   - Coeficientes del sistema.
   - Términos independientes.
   - Valores iniciales.
   - Tolerancia.
   - Número máximo de iteraciones.

5. Revisar los resultados mostrados en consola.

- Jacobi.java
- GaussSeidel.java
