# Programas - Unidad 3

## Descripción

Esta carpeta contiene los programas desarrollados en Java para resolver sistemas de ecuaciones lineales mediante métodos iterativos.

## Archivos incluidos

### Jacobi.java

Programa que aplica el Método de Jacobi para resolver sistemas de ecuaciones lineales.  
El usuario ingresa el número de ecuaciones, los coeficientes del sistema, los términos independientes, los valores iniciales, la tolerancia y el número máximo de iteraciones.

El programa muestra las aproximaciones obtenidas en cada iteración y el error mayor hasta alcanzar la tolerancia indicada.

### GaussSeidel.java

Programa que aplica el Método de Gauss-Seidel para resolver sistemas de ecuaciones lineales.  
El usuario ingresa los datos del sistema y el programa calcula nuevas aproximaciones utilizando los valores más recientes en cada iteración.

El programa muestra los resultados de cada iteración y finaliza cuando se cumple la tolerancia establecida.

## Objetivo

Automatizar la solución de sistemas de ecuaciones lineales mediante los métodos de Jacobi y Gauss-Seidel.

## Lenguaje utilizado

- Java
