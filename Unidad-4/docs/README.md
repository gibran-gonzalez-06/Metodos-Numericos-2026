# Documentación - Unidad 4

## Descripción

Esta carpeta contiene las evidencias y actividades desarrolladas durante la Unidad 4 de la asignatura Métodos Numéricos.

## Contenido

- Evidencias fotográficas de ejercicios realizados en libreta.
- Procedimientos desarrollados para la resolución de problemas de integración numérica.
- Actividades académicas correspondientes a la unidad.

## Temas trabajados

- Método del Trapecio.
- Método de Simpson 1/3.
- Aproximación numérica de integrales definidas.
- Aplicación de métodos de integración mediante programación en Java.

## Objetivo

Documentar el desarrollo de ejercicios y actividades realizadas durante la Unidad 4 para reforzar la comprensión de los métodos de integración numérica.
