# Entorno de Desarrollo - Unidad 4

## Descripción

Esta carpeta documenta el entorno utilizado para el desarrollo y ejecución de los programas correspondientes a la Unidad 4.

## Software utilizado

- Java
- NetBeans IDE
- JDK (Java Development Kit)

## Equipo de trabajo

- Laptop HP
- Sistema Operativo Windows

## Propósito

Contar con un entorno adecuado para implementar y ejecutar los métodos numéricos de integración desarrollados durante la unidad.

## Métodos implementados

- Método del Trapecio
- Método de Simpson 1/3
