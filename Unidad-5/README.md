# Unidad 5 - Interpolación y Ajuste de Funciones

## Descripción

En esta unidad se estudiaron métodos numéricos utilizados para aproximar valores intermedios a partir de un conjunto de datos conocidos. Se trabajó principalmente con el método de interpolación de Lagrange.

## Objetivo

Aplicar métodos de interpolación para estimar valores desconocidos mediante el uso de polinomios construidos a partir de datos conocidos.

## Actividades realizadas

- Resolución de ejercicios del cuadernillo.
- Aplicación del método de interpolación de Lagrange.
- Revisión de ejercicios de diferencias divididas de Newton.
- Análisis de mínimos cuadrados y correlación.
- Desarrollo de un programa en Java para el método de Lagrange.

## Programa desarrollado

### MetodoLagrange.java

Programa que permite calcular el valor interpolado de una función utilizando el método de Lagrange. El usuario ingresa los puntos conocidos y el valor de x que desea interpolar.

## Temas trabajados

- Interpolación de Lagrange.

- Diferencias divididas de Newton.

- Mínimos cuadrados.

- Correlación.
