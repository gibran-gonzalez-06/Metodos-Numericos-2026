# Evidencias de Ejecución - Unidad 5

## Descripción

Esta carpeta contiene la evidencia de ejecución del programa MetodoLagrange.java, correspondiente al Método de Interpolación de Lagrange.

## Evidencia incluida

Debido a la cantidad de capturas generadas durante la ejecución del programa, las imágenes fueron integradas en un solo archivo PDF para facilitar su revisión y mantener una mejor organización dentro del repositorio.

El archivo PDF muestra el proceso completo de ejecución del programa, incluyendo:

- Ingreso de la cantidad de puntos.
- Captura de los valores de x.
- Captura de los valores de y.
- Ingreso del valor a interpolar.
- Desarrollo del procedimiento del Método de Lagrange.
- Resultado final obtenido.

## Ejemplo de prueba

Puntos utilizados:

- (1,1)
- (2,4)
- (3,9)

Valor a interpolar:

- x = 2.5

Resultado obtenido:

- P(2.5) = 6.25

## Objetivo

Comprobar el funcionamiento correcto del programa y documentar de forma ordenada el procedimiento realizado durante la ejecución.
