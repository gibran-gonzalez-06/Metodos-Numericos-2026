# Documentación - Unidad 6

## Descripción

Esta carpeta contiene la documentación correspondiente a las actividades realizadas durante la Unidad 6.

## Contenido

- Ejercicios resueltos en libreta.
- Procedimientos desarrollados paso a paso.
- Aplicación de los métodos numéricos estudiados.
- Evidencias académicas relacionadas con ecuaciones diferenciales ordinarias.

## Temas incluidos

- Método de Euler.
- Método de Euler Mejorado (Heun).
- Problemas de valor inicial.
- Aproximación numérica de soluciones.

## Objetivo

Documentar los procedimientos y actividades desarrollados durante la unidad para reforzar la comprensión de los métodos numéricos aplicados a ecuaciones diferenciales.
