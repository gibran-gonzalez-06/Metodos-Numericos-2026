# Código Fuente - Unidad 6

## Descripción

Durante el desarrollo de la Unidad 6 no fue necesario implementar programas o aplicaciones de software.

Las actividades realizadas estuvieron enfocadas en la resolución manual de ejercicios relacionados con ecuaciones diferenciales ordinarias y métodos numéricos, desarrollados directamente en libreta como parte de las actividades académicas de la unidad.

## Observación

No se generó código fuente para esta unidad, por lo que esta carpeta se mantiene únicamente con fines de organización y para conservar la estructura general del repositorio.

## Temas trabajados

- Método de Euler.
- Método de Euler Mejorado (Heun).
- Solución aproximada de ecuaciones diferenciales ordinarias.
- Problemas de valor inicial.

## Estado

No aplica desarrollo de software para esta unidad.
