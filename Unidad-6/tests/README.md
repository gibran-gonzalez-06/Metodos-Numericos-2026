# Evidencias - Unidad 6

## Descripción

Durante esta unidad no se desarrollaron programas computacionales ni aplicaciones de software que requirieran pruebas de ejecución.

Las actividades consistieron en la resolución manual de ejercicios relacionados con ecuaciones diferenciales ordinarias y métodos numéricos, por lo que las evidencias correspondientes se encuentran integradas en la carpeta docs.

## Observación

No aplica la generación de casos de prueba o evidencias de ejecución debido a la naturaleza teórica y práctica de la unidad.
